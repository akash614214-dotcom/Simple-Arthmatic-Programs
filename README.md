# 📘 C Programming Practice Collection

A collection of beginner-friendly C programming programs created to strengthen the fundamentals of programming, problem-solving, arrays, loops, mathematics, and basic algorithms.

---

## 📂 Programs Included

### 1️⃣ Simple Array Programs
A collection of basic array operations including:
- Array Input & Output
- Sum of Array Elements
- Average of Elements
- Searching
- Reversing an Array
- Other beginner array operations

---

### 2️⃣ Square Program
Program to calculate:
- Square of a Number
- Mathematical operations related to squares

---

### 3️⃣ Area of Circle & Sum of First N Numbers
This program performs:
- Calculate Area of a Circle
- Calculate the Sum of First N Natural Numbers

**Formula Used**
- Area = π × r²
- Sum = n(n + 1) / 2

---

### 4️⃣ Found Elements
Program to:
- Search an element in an array.
- Display whether the element is found or not.
- Can be implemented using Linear Search.

---

### 5️⃣ Largest Number in Array
Program to:
- Find the largest element in an array.
- Demonstrates array traversal and comparison logic.
